# 手工艺人派单接单小程序

一个基于微信小程序云开发的手工艺人派单接单系统，支持管理员派发订单和手工艺人接单功能。

## 功能特性

### 管理员端
- 📋 订单管理：创建、编辑、取消、删除订单
- 🎨 样式管理：管理订单样式，支持新建和复用
- 👨‍🎨 匠人管理：管理手工艺人信息，包括星级、擅长方向、履约情况
- 📊 数据统计：查看订单统计信息

### 手工艺人端
- 🏪 接单大厅：浏览待接单列表，抢单接单
- 📦 我的订单：查看已接订单，完成订单
- 👤 个人中心：查看个人信息和接单统计
- 🔔 消息提醒：新订单微信订阅消息通知

### 订单信息
- 订单名称
- 样式（支持下拉选择或新建）
- 数量
- 单价/总价
- 派单日期
- 收货日期
- 备注

### 手工艺人信息
- 姓名
- 星级（1-5星）
- 擅长方向
- 履约情况
- 累计接单数
- 已完成订单数

## 技术栈

- 微信小程序原生开发
- 微信云开发（云数据库 + 云函数）
- 微信订阅消息

## 快速开始

### 1. 环境准备

1. 注册微信小程序账号
2. 开通微信云开发
3. 下载安装微信开发者工具

### 2. 项目配置

1. 使用微信开发者工具导入项目
2. 修改 `project.config.json` 中的 `appid` 为你自己的小程序 AppID
3. 修改 `miniprogram/app.js` 中的云开发环境ID：
   ```javascript
   wx.cloud.init({
     env: 'your-env-id', // 替换为你的云开发环境ID
     traceUser: true,
   })
   ```

### 3. 配置管理员

修改 `cloudfunctions/craftsman/index.js` 中的管理员openid：

```javascript
const ADMIN_OPENIDS = ['o YOUR_ADMIN_OPENID_HERE']
```

获取openid的方法：临时部署项目，在任意页面调用 `wx.cloud.callFunction` 获取。

### 4. 部署云函数

在微信开发者工具中：
1. 右键 `cloudfunctions/order` 选择 "创建并部署：云端安装依赖"
2. 右键 `cloudfunctions/style` 选择 "创建并部署：云端安装依赖"
3. 右键 `cloudfunctions/craftsman` 选择 "创建并部署：云端安装依赖"
4. 右键 `cloudfunctions/message` 选择 "创建并部署：云端安装依赖"
5. 右键 `cloudfunctions/subscribe` 选择 "创建并部署：云端安装依赖"

### 5. 创建数据库集合

在云开发控制台的数据库中创建以下集合：
- `orders` - 订单数据
- `styles` - 样式数据
- `craftsmen` - 手工艺人数据
- `subscribers` - 订阅数据
- `notices` - 公告数据（可选）

为各集合设置合适的权限（建议：
- `orders`：所有用户可读，仅创建者可写
- `styles`：所有用户可读，仅管理员可写
- `craftsmen`：所有用户可读，仅管理员可写
- `subscribers`：仅用户自己可读写
- `notices`：所有用户可读，仅管理员可写

### 6. 配置订阅消息

1. 登录微信公众平台
2. 进入 "功能" → "订阅消息"
3. 添加以下模板：
   - 新订单提醒
   - 接单成功提醒
   - 订单完成提醒
4. 将模板ID更新到 `cloudfunctions/message/index.js` 中的 `TMPL_IDS`

### 7. 运行项目

点击微信开发者工具的 "编译" 按钮即可运行。

## 使用说明

### 管理员使用流程

1. 首次使用需要配置管理员openid
2. 进入小程序后，点击 "样式管理" 创建常用的订单样式
3. 点击 "匠人管理" 添加手工艺人信息
4. 点击 "新建订单" 创建派单订单
5. 手工艺人将会收到新订单提醒

### 手工艺人使用流程

1. 联系管理员在 "匠人管理" 中注册
2. 进入小程序，在 "接单大厅" 浏览可接订单
3. 点击 "立即接单" 抢单
4. 在 "我的订单" 中查看已接订单
5. 制作完成后点击 "完成订单"

## 注意事项

1. 订阅消息需要用户主动授权才能发送
2. 管理员openid需要在云函数中配置
3. 手工艺人需要先由管理员注册才能接单
4. 建议定期备份云数据库数据
