## 四、常见问题解决

### ❓ 验证码发送失败
**排查**：
1. 检查 `sms` 云函数是否已部署
2. 检查数据库集合 `verifyCodes` 是否存在
3. 查看云函数调用日志

### ❓ 手艺人无法接单
**排查**：
1. 确认手艺人已通过审核（status = 'active'）
2. 检查 `app.globalData.craftsmanInfo` 是否有值
3. 查看订单状态是否为 'pending'

### ❓ 没有管理员入口
**排查**：
1. 确认使用的是手机号 13810062394
2. 检查 `users` 集合中 role 字段是否为 'admin'
3. 尝试重新编译小程序

### ❓ 订单列表为空
**排查**：
1. 确认已调用 `initTestData` 云函数创建测试订单
2. 检查 `orders` 集合中是否有数据
3. 确认订单状态为 'pending'

## 五、清理测试数据

测试完成后，执行以下代码清理数据：

```javascript
// 清理测试数据
wx.cloud.callFunction({
  name: 'initTestData',
  data: { action: 'clearTestData' }
})
```

**注意**：此操作会删除所有订单数据，请谨慎使用！
