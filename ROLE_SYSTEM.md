# 角色系统和验证码验证说明

## 功能概述

小程序已实现三角色系统和验证码验证机制：

| 角色 | 权限 | 注册方式 |
|------|------|----------|
| 🎨 **手艺人** | 接单、查看订单、管理个人订单 | 手机号验证码注册，需审核 |
| 📋 **派单人** | 发布订单、管理订单、查看手艺人 | 手机号验证码验证 |
| ⚙️ **管理员** | 管理所有用户、审核注册、删除用户 | 手机号 13810062394 自动成为管理员 |

## 数据库集合

| 集合名 | 用途 |
|--------|------|
| users | 存储用户基本信息和角色 |
| craftsmen | 存储手艺人详细信息 |
| dispatchers | 存储派单人认证信息 |
| verifyCodes | 存储验证码（10分钟有效） |
| smsLogs | 存储短信发送日志 |
| adminRequests | 存储管理员审核请求 |

## 配置步骤

### 1. 修改云开发环境ID

编辑 `app.js`，替换 `your-env-id`：

```javascript
wx.cloud.init({
  env: '你的云开发环境ID',  // ← 替换为您的云开发环境ID
  traceUser: true,
})
```

### 2. 部署云函数

在微信开发者工具中依次部署以下云函数：

```bash
# 1. 初始化云函数
右键 cloudfunctions/init → 创建并部署：云端安装依赖

# 2. 用户管理云函数
右键 cloudfunctions/user → 创建并部署：云端安装依赖

# 3. 短信验证码云函数
右键 cloudfunctions/sms → 创建并部署：云端安装依赖

# 4. 管理员后台云函数
右键 cloudfunctions/admin → 创建并部署：云端安装依赖
```

### 3. 初始化数据库

调用初始化云函数创建所有集合：

```javascript
// 在云开发控制台或代码中调用
wx.cloud.callFunction({
  name: 'init'
})
```

## 验证码验证机制

### 手艺人注册流程

1. 填写姓名、手机号等信息
2. 点击"获取验证码"
3. 系统生成 6 位验证码，保存到数据库，有效期 10 分钟
4. **同时向管理员手机 13810062394 发送通知**（包含验证码）
5. 用户输入收到的验证码
6. 系统验证通过后，创建手艺人记录
7. **等待管理员审核**（非 13810062394 手机号）
8. 审核通过后即可接单

### 派单人认证流程

1. 填写手机号、姓名等信息
2. 点击"获取验证码"
3. 系统生成 6 位验证码
4. **向管理员手机 13810062394 发送通知**（包含验证码）
5. 用户输入收到的验证码
6. 系统验证通过后，创建派单人记录
7. 立即可发布订单

### 验证码接口

```javascript
// 发送验证码
wx.cloud.callFunction({
  name: 'sms',
  data: {
    action: 'sendVerifyCode',
    data: {
      phone: '13800138000',
      type: 'craftsman'  // 或 'dispatcher'
    }
  }
})

// 验证验证码
wx.cloud.callFunction({
  name: 'sms',
  data: {
    action: 'verifyCode',
    data: {
      phone: '13800138000',
      code: '123456',
      type: 'craftsman'
    }
  }
})
```

## 管理员功能

### 自动识别管理员

- 使用手机号 **13810062394** 注册手艺人或派单人时，自动成为管理员
- 无需审核，立即拥有管理员权限

### 管理员控制台

访问路径：`/pages/admin/console`

功能包括：
- 📊 **数据统计**：查看手艺人、派单人、待审核数量
- 👨‍🎨 **手艺人管理**：查看、删除手艺人
- 📋 **派单人管理**：查看、删除派单人
- ✅ **审核管理**：审核手艺人注册申请

### 管理员权限

```javascript
// 检查是否为管理员
const res = await wx.cloud.callFunction({
  name: 'user',
  data: { action: 'isAdmin' }
})

if (res.result.isAdmin) {
  // 执行管理员操作
}
```

## 用户管理

### 删除用户

管理员可以删除手艺人或派单人：

```javascript
// 删除手艺人
wx.cloud.callFunction({
  name: 'admin',
  data: {
    action: 'deleteCraftsman',
    data: { craftsmanId: 'xxx' }
  }
})

// 删除派单人
wx.cloud.callFunction({
  name: 'admin',
  data: {
    action: 'deleteDispatcher',
    data: { dispatcherId: 'xxx' }
  }
})
```

**注意**：不能删除管理员账号（手机号 13810062394）

## 安全说明

1. **验证码安全**：
   - 有效期 10 分钟
   - 使用后立即失效
   - 1 分钟内不能重复发送

2. **权限控制**：
   - 所有管理员操作都经过权限验证
   - 不能删除管理员账号
   - 一个手机号只能注册一种角色

3. **数据存储**：
   - 所有用户信息存储在云数据库
   - 验证码过期后自动清理（需要配置定时触发器）
