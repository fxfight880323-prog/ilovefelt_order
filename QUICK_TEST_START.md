## 常见问题快速解决

### 问题1：数据库权限错误

**现象：** `database permission denied`

**解决：**
1. 打开云开发控制台
2. 进入数据库 → 权限设置
3. 设置为 "所有用户可读可写"
4. 等待1分钟后重试

### 问题2：云函数调用失败

**现象：** `cloud.callFunction error`

**解决：**
1. 右键 `cloudfunctions/api` 
2. 选择 "创建并部署：云端安装依赖"
3. 等待部署完成

### 问题3：登录提示未审批

**现象：** 登录提示 "账号正在审核中"

**解决：**
1. 用管理员账号登录
2. 进入控制台审批该用户
3. 重新登录

### 问题4：看不到订单

**现象：** 接单大厅为空

**检查：**
```javascript
// 在控制台查看订单
wx.cloud.database().collection('orders').get().then(res => {
  console.log('订单数量:', res.data.length)
  console.log('订单状态:', res.data.map(o => o.status))
})
```

**解决：**
- 如果没有订单，用派单人账号创建
- 如果订单status不是pending，检查数据
