## 测试数据清理

测试完成后如需清理数据：

```javascript
const db = wx.cloud.database()

// 清理测试订单
db.collection('orders').where({
  name: db.RegExp({ regexp: '测试|正式' })
}).remove()

// 清理测试用户
db.collection('users').where({
  phone: db.command.in(['13800138001', '13800138002'])
}).remove()

// 清理角色记录
db.collection('craftsmen').where({
  phone: '13800138002'
}).remove()
db.collection('dispatchers').where({
  phone: '13800138001'
}).remove()
```

## 注意事项

1. **首次运行前**：确保超级管理员 13810062394/880323 已创建
2. **云函数部署**：确保 `cloudfunctions/api` 已部署
3. **数据库权限**：确保所有集合可读写
4. **重复运行**：测试数据会累积，建议定期清理

## 测试失败排查

### 问题1: "账号或密码错误"
- 检查超级管理员密码是否正确
- 查看数据库中的实际密码值

### 问题2: "无权限访问"
- 确保超级管理员角色包含 'admin'
- 检查 isSuperAdmin 字段是否为 true

### 问题3: "订单创建失败"
- 确保派单人已通过审批
- 检查登录状态是否正确

### 问题4: "接单失败"
- 确保手艺人已通过审批
- 确保订单状态为 'pending'

## 分步调试
