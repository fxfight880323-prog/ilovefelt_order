# 登录页注册功能更新

## 修改内容

### 1. 登录页面 (`pages/login/index`)

**新增"注册新账号"按钮**

**WXML 修改：**
- 移除了"账号密码登录"按钮
- 添加了"注册新账号"按钮（橙色渐变样式）

**JS 修改：**
- 添加 `goToRegister()` 方法
- 点击后弹出选择框："注册为手艺人"或"注册为派单人"
- 选择后跳转到对应注册页面

**界面效果：**
```
📤 微信一键登录
📝 注册新账号  ← 橙色按钮

      暂不登录，随便看看 >
```

点击"注册新账号"后弹出：
- 注册为手艺人 → 跳转到手艺人注册页
- 注册为派单人 → 跳转到派单人认证页

### 2. 管理员权限控制 (`cloudfunctions/user/index.js`)

**管理员手机号设置：**
```javascript
const ADMIN_PHONE = '13810062394'
```

**注册逻辑：**
- 手机号为 `13810062394` 时：
  - 自动通过审核（无需等待）
  - 自动设为管理员角色
  - 可以同时拥有手艺人/派单人 + 管理员角色
  
- 其他手机号：
  - 提交后需要管理员审核
  - 不会自动成为管理员

## 注册流程

### 普通用户注册
```
登录页 → 点击注册 → 选择角色 → 填写表单 → 提交申请 → 等待审核
```

### 管理员注册（13810062394）
```
登录页 → 点击注册 → 选择角色 → 填写表单(手机13810062394) → 提交 → 立即通过 → 自动成为管理员
```
