# 图标资源

小程序需要使用以下图标文件，请准备 64x64 像素的 PNG 图片：

## TabBar 图标

| 文件名 | 用途 | 建议颜色 |
|--------|------|----------|
| home.png | 首页-未选中 | #999999 |
| home-active.png | 首页-选中 | #1890ff |
| order.png | 接单-未选中 | #999999 |
| order-active.png | 接单-选中 | #1890ff |
| my-order.png | 我的订单-未选中 | #999999 |
| my-order-active.png | 我的订单-选中 | #1890ff |
| profile.png | 个人中心-未选中 | #999999 |
| profile-active.png | 个人中心-选中 | #1890ff |

## 获取图标

可以使用以下方式获取图标：

### 方式一：使用在线图标库
1. 访问 [iconfont](https://www.iconfont.cn/)
2. 搜索需要的图标（如：首页、订单、我的）
3. 下载 PNG 格式，颜色改为上述建议颜色

### 方式二：使用微信官方图标
可以从小程序设计资源中获取标准图标

### 方式三：简单文字替代
开发测试阶段，可以先用文字代替，后续再替换为图标
