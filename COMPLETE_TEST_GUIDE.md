## 第一阶段：环境初始化测试

### 测试 1.1：数据库权限测试

**步骤：**
1. 在微信开发者工具控制台运行：
```javascript
// 测试读取权限
wx.cloud.database().collection('users').get().then(res => {
  console.log('✅ 数据库读取权限正常', res.data)
}).catch(err => {
  console.error('❌ 数据库读取权限失败', err)
})

// 测试写入权限
wx.cloud.database().collection('users').add({
  data: { 
    test: true, 
    createTime: new Date() 
  }
}).then(res => {
  console.log('✅ 数据库写入权限正常', res._id)
  // 清理测试数据
  wx.cloud.database().collection('users').doc(res._id).remove()
}).catch(err => {
  console.error('❌ 数据库写入权限失败', err)
})
```

**预期结果：**
- 读取成功，返回空数组或已有数据
- 写入成功，返回新记录ID

### 测试 1.2：云函数部署测试

**步骤：**
```javascript
// 测试云函数是否正常运行

}).then(res => {
  console.log('✅ 云函数正常', res.result)
}).catch(err => {
  console.error('❌ 云函数调用失败', err)
})
```

**预期结果：**
- 返回 `{ success: true/false, data: {...}, msg: '...' }`

2. 点击 "注册新账号" → "注册为手艺人"
3. 填写表单：

   - 手机号：13800138001
   - 验证码：（从控制台获取或跳过）
   - 擅长工艺：木工
   - 从业经验：3-5年
