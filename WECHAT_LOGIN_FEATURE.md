# 微信一键登录功能实现

## 功能概述

实现了微信授权登录功能，支持手机号绑定微信，实现一键登录。

## 实现内容

### 1. 云函数更新 (cloudfunctions/api/index.js)

#### 新增接口

**auth.wechatLogin** - 微信一键登录
- 检查当前微信openid是否已绑定用户
- 如果已绑定，直接返回用户信息完成登录
- 如果未绑定，返回`bound: false`，前端显示绑定手机号弹窗

**auth.bindPhone** - 绑定手机号到微信
- 验证手机号和密码
- 检查是否已被其他微信绑定
- 绑定openid到用户账号
- 返回用户信息完成登录

### 2. 登录页更新 (pages/login/index.js)

#### 新增功能

- `onWechatLogin()` - 微信一键登录处理
- `onBindPhone()` - 绑定手机号处理
- `closeBindModal()` - 关闭绑定弹窗
- `onBindPhoneInput()` - 绑定手机号输入
- `onBindPasswordInput()` - 绑定密码输入

### 3. 登录页UI更新 (pages/login/index.wxml)

新增绑定手机号弹窗，包含：
- 手机号输入框
- 密码输入框
- 取消/确认按钮

### 4. 样式更新 (pages/login/index.wxss)

新增绑定弹窗样式：
- 弹窗遮罩
- 居中内容区域
- 表单样式
- 按钮样式

### 5. API工具更新 (utils/api.js)

新增方法：
- `API.auth.wechatLogin()` - 微信一键登录
- `API.auth.bindPhone(data)` - 绑定手机号

## 使用流程

### 已绑定用户
1. 点击"微信一键登录"
2. 自动获取微信openid
3. 如果已绑定，直接登录成功，跳转到首页

### 未绑定用户
1. 点击"微信一键登录"
2. 自动获取微信openid
3. 如果未绑定，弹出"绑定手机号"弹窗
4. 输入已有账号的手机号和密码
5. 点击"立即绑定"
6. 绑定成功，自动登录，跳转到首页

## 数据模型更新

用户集合(users)新增字段：
- `openid` - 微信openid
- `bindWechatTime` - 绑定时间
- `lastLoginTime` - 最后登录时间

## 注意事项

1. 微信登录需要在真机调试或预览模式下测试
2. 开发者工具的模拟器无法获取真实的openid
3. 每个微信openid只能绑定一个账号
4. 一个账号只能绑定一个微信openid
