# 登录页使用说明

## 图片设置

请将提供的 Logo 图片保存到项目目录下：
```
images/logo.jpg
```

图片要求：
- 建议尺寸：400x400 像素以上
- 格式：JPG
- 背景：透明或浅色

## 页面功能

### 1. 启动页显示
- Logo 至少显示 2 秒
- 显示品牌名称和标语
- 加载动画效果

### 2. 自动登录检测
- 检查是否已登录
- 已登录用户自动跳转到首页
- 未登录显示登录按钮

### 3. 登录方式
- **微信一键登录**：使用微信授权登录
- **账号密码登录**：待开发功能
- **游客进入**：不登录浏览部分内容

### 4. 协议同意
- 登录前需要勾选同意用户协议
- 可查看《用户服务协议》和《隐私政策》

## 页面路径

```
pages/login/index
```

## 已更新配置

1. app.json - 将登录页设为首页
2. app.js - 移除启动时自动登录检查
3. 云函数 user - 支持 login 和 getUserInfo

## 调试日志

在微信开发者工具控制台查看：
- "登录页 onLoad"
- "正在进入..."（已登录）
- "登录成功"（登录成功）
