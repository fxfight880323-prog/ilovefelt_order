# 匠人管理工作记录功能

## 功能概述

已扩展匠人管理功能，支持：
- ✅ 制作师代号管理
- ✅ 制作成本记录
- ✅ 产品图上传
- ✅ 交货时间记录
- ✅ 完成状态（已完成/返工/库存）
- ✅ 结算金额管理

## 数据结构

### craftsmen 集合新增字段
```javascript
{
  code: 'A001',  // 制作师代号
  // 其他原有字段...
}
```

### craftsmanWorks 集合（新增）
```javascript
{
  craftsmanId: '手艺人ID',
  craftsmanName: '手艺人姓名',
  craftsmanCode: '制作师代号',
  orderId: '关联订单ID',
  orderName: '订单/产品名称',
  productImage: '产品图片URL',
  costPrice: 100.00,           // 制作成本
  settlementAmount: 150.00,    // 结算金额
  deliveryDate: '2024-03-20',  // 交货时间
  completionStatus: 'completed', // 完成状态
  remarks: '备注',
  createTime: '创建时间',
  updateTime: '更新时间'
}
```

## 界面说明

### 匠人列表
- 显示制作师代号（如有）
- 点击"详情"查看完整信息和工作记录

### 基本信息页
- 姓名、制作师代号、手机号
- 擅长方向、星级
- 累计接单数、已完成数

### 工作记录页
- **添加工作记录**：
  - 上传产品图片
  - 填写订单/产品名称
  - 输入制作成本
  - 输入结算金额
  - 选择交货时间
  - 选择完成状态（已完成/返工/库存）
  - 填写备注

- **工作记录列表**：
  - 显示产品缩略图
  - 显示订单名称
  - 显示完成状态标签
  - 显示成本（红色）和结算金额（绿色）
  - 显示交货时间和备注
  - 可删除记录

## 完成状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 已完成 | 绿色 | 订单正常完成 |
| 返工 | 红色 | 需要重新制作 |
| 库存 | 黄色 | 作为库存保留 |

## 使用流程

### 添加匠人
1. 点击右下角"+"按钮
2. 填写基本信息（姓名、代号、手机号等）
3. 选择星级和擅长方向
4. 点击保存

### 添加工作记录
1. 在匠人列表点击"详情"
2. 切换到"工作记录"标签
3. 点击"添加工作记录"
4. 填写工作信息（产品图、成本、结算金额等）
5. 选择完成状态
6. 点击保存

### 查看统计
- 在详情页查看基本信息
- 在工作记录页查看所有历史记录
- 可查看成本、结算金额对比

## 部署说明

### 需要部署的云函数
```
cloudfunctions/craftsman       - 更新：添加 code 字段支持
cloudfunctions/craftsmanWorks  - 新增：工作记录管理
cloudfunctions/init            - 更新：添加 craftsmanWorks 集合
```

### 部署步骤
1. 右键 `cloudfunctions/craftsman` → 创建并部署
2. 右键 `cloudfunctions/craftsmanWorks` → 创建并部署
3. 右键 `cloudfunctions/init` → 创建并部署（更新数据库）
4. 重新编译小程序

## 注意事项

1. **制作师代号**：建议按规则命名（如A001、B002）
2. **成本与结算**：成本为制作花费，结算为实际收入
3. **产品图**：支持上传多张，但目前只显示第一张
4. **权限**：只有管理员可添加/编辑工作记录
