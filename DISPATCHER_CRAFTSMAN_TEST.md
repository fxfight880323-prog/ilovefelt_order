在微信开发者工具的「调试器」→「Console」控制台中运行：

```javascript

// 运行完整测试

```

### 方式二：测试页面

1. 确保已在 `app.json` 中注册测试页面（已自动添加）
2. 在微信开发者工具中编译项目
3. 通过扫码或跳转到测试页面：

## 常见问题

### 1. 超级管理员登录失败
确保超级管理员账号 `13810062394/880323` 存在于数据库中。

### 2. 审批失败
如果API审批失败，测试会自动通过数据库操作完成审批。

### 3. 数据库连接失败
检查云开发环境是否正常，数据库权限是否正确设置。

### 4. 账号已注册
如果测试账号已注册，测试会跳过注册步骤直接登录。
