## 数据库验证命令

### 验证账号A的数据隔离
```javascript
// 查看账号A的订单
const phoneA = '13800000001'
const userA = await db.collection('users').where({ phone: phoneA }).get()
const craftsmanA = await db.collection('craftsmen').where({ phone: phoneA }).get()

if (craftsmanA.data.length > 0) {
  const craftsmanIdA = craftsmanA.data[0]._id
  const ordersA = await db.collection('orders').where({ 
    craftsmanId: craftsmanIdA 
  }).get()
  console.log('账号A的订单:', ordersA.data.map(o => o._id))
}
```

### 验证不同账号数据不重叠
```javascript
// 确保A和B的订单不重叠
const ordersAIds = ordersA.data.map(o => o._id)
const ordersBIds = ordersB.data.map(o => o._id)
const overlap = ordersAIds.filter(id => ordersBIds.includes(id))
console.log('重叠订单:', overlap) // 应该是空数组
```
